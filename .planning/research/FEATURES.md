# Features & Curriculum: DSA Mastery

## Core Topics (Wengrow Method)

Based on *A Common-Sense Guide to Data Structures and Algorithms*, the curriculum should be organized by data structure and algorithmic pattern.

### 1. Fundamentals of Efficiency
- **Big O Notation**: Understanding "Steps, not Seconds."
- **Constant, Linear, and Quadratic Time**: The primary "stories" of Big O.

### 2. Linear Data Structures
- **Arrays**: Reading, Searching, Insertion, Deletion (The "Parking Lot").
- **Ordered Arrays**: Binary Search (The "Dictionary Game").
- **Sets**: Uniqueness constraints.
- **Linked Lists**: The "Paper Chain" analogy.
- **Doubly Linked Lists**: Two-way traversal.

### 3. Hash-Based Structures
- **Hash Tables**: The "Magical Encyclopedia."
- **Dealing with Collisions**: Chaining and efficiency.

### 4. Hierarchical & Complex Structures
- **Binary Search Trees (BST)**: Searching, Insertion, Deletion.
- **Heaps**: Priority Queues.
- **Tries**: Autocomplete and Prefix matching.
- **Graphs**: Social networks and mapping.

### 5. Algorithmic Patterns
- **Recursion**: "Kicking the problem down the road."
- **Dynamic Programming**: Memoization and overlapping subproblems.
- **Sorting**: Bubble, Selection, Insertion, QuickSort, Merge Sort.
- **Space Complexity**: Memory vs. Time tradeoffs.

## User-Requested Tracking Features
- **Status Toggles**: Solved (Green), Attempted (Yellow), Need to Revisit (Red/Gold).
- **Persistence**: Save state automatically to `localStorage`.
- **Direct Links**: Button to open LeetCode directly for each problem.
- **Problem Details**: Expanded view showing *why* this problem relates to the topic.
